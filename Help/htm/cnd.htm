SDL Joystick Object - Conditions

Device is connected
Returns true if a device is connected. Device IDs range from 0 to 15.

Button is pressed
Returns true if a specific button on a device is pressed. Device IDs range from 0 to 15, button IDs range from 0 to 31.

Button is held down
Returns true if a specific button on a device is held down. Device IDs range from 0 to 15, button IDs range from 0 to 31.

Button is released
Returns true if a specific button on a device is no longer held down. Device IDs range from 0 to 15, button IDs range from 0 to 31.

Any button is pressed
Returns true if any button on a device is pressed. Device IDs range from 0 to 15.

Any button is held down
Returns true if any button on a device is held down. Device IDs range from 0 to 15.

Any button is released
Returns true of any button on a device is no longer held down. Device IDs range from 0 to 15.
