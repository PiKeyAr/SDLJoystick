SDL Joystick Object

SDL Joystick is an extension that uses the Joystick API of the SDL2 library to interface with game controllers. It can work with up to 16 devices connected simultaneously. Each device supports up to:

8 analog axes
32 buttons
4 POV hats
4 trackballs

SDL GameController abstraction and the accompanying mapping system are not supported at the moment.

The latest version of this extension is available on GitHub along with the source code.

SDL Joystick uses the SDL2 library, and you must conform to its license terms in all projects using this extension.
